### 2. 数据最小化原则
```
采集的数据:
✅ 收藏的 URL/ID
✅ 收藏时间
✅ 标题/描述
✅ 分类/标签

不采集的数据:
❌ 账号密码
❌ 浏览历史
❌ 聊天记录
❌ 个人信息 (姓名、电话、地址)
❌ 设备信息
```

### 3. 零知识证明
```
即使数据被窃取:
- 所有数据 AES-256 加密
- 密钥只在你脑中 (密码短语)
- 我们无法解密你的数据
- 攻击者也无法解密

即使我们被要求提供数据:
- 只能提供加密的乱码
- 没有解密密钥
- 法律上无法配合
```

### 4. 透明可控
```
你完全控制:
✅ 可以看到所有采集的数据
✅ 可以删除任意内容
✅ 可以导出全部数据
✅ 可以完全清空
✅ 可以选择分析哪些平台
✅ 可以随时停止使用

我们承诺:
✅ 不开启云端分析
✅ 不收集使用统计
✅ 不出售任何数据
✅ 不做用户画像广告
✅ 开源核心代码 (可审计)
```

### 1. 数据采集层

# 统一的数据采集接口
class DataSource(ABC):
    """数据源基类"""
    
    def __init__(self, name: str, privacy_level: str):
        self.name = name
        self.privacy_level = privacy_level  # low/medium/high
    
    @abstractmethod
    def fetch(self, credentials: Dict) -> List[Item]:
        """获取收藏数据"""
        pass
    
    @abstractmethod
    def parse(self, raw_data: Any) -> Item:
        """解析为统一格式"""
        pass

# 各平台实现
class BrowserBookmarks(DataSource):
    def fetch(self):
        # 读取本地 Chrome JSON
        # 无需网络，隐私风险低
        pass

class GitHubStars(DataSource):
    def fetch(self, token: str):
        # GitHub API (公开数据)
        # 使用 Personal Access Token
        pass

class ZhihuFavorites(DataSource):
    def fetch(self, cookies: str):
        # 知乎爬虫 (需要登录)
        # 隐私风险中等
        pass

class BilibiliFavorites(DataSource):
